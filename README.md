# Overview
This is a Flashcard app that runs on the console. It allow the user to create stacks of flashcards with questions and answers and use them to study and memorize it's contents. It's the [Flashcard app](https://www.thecsharpacademy.com/project/14/flashcards) project for the C# academy.

# Features
- Scoring system to evaluate how well you did on each try
- Persistence of data locally with SQL server database
- Reports to review and compare past activities

# Installation
- Clone the repository `git clone https://github.com/diegopetrola/CodeReviews.Console.Flashcards` and go to it's folder
- Install packages, on Visual Studio packet manager `Restore`
- Add misgratioons and create the database `Add-Migrations InitialMigration` and `Update-Database`
- Run the project `dotnet run`
